## ðŸ“¦ Source Code (`src/`)
- `components/`:
    - `layout/`: Main app shell (`Dashboard`, `Sidebar`).
    - `common/`: Reusable UI primitives.
- `context/`: React context providers for global services (Audio, MIDI).
- `hooks/`: Shared React hooks for common logic.
- `modules/`: Feature-complete musical tools (The heart of the app).
    - `ChordLab/`: Core chord progression builder.
    - `JazzKiller/`: Jazz standards analysis and playback.
    - `FunctionalEarTraining/`: Interactive ear training levels.
    - `GripSequencer/`: Visual guitar voicing sequencer.
    - `CircleChords/`: Circle of Fifths visualization.
    - `Tonnetz/`: 3D Tonnetz geometric navigator.
- `pages/`: Page-level components for simpler views (e.g., `MidiLibraryPage`).
- `utils/`: Core musical logic, engine utilities, and theory calculations.
